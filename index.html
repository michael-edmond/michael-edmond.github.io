Security-Aware Automation Projects

These projects demonstrate the design and deployment of secure, automated systems for real-world school operations, emphasizing access control, auditability, and operational integrity.

ClassTrack- Secure School Attendance Automation System

Designed and deployed a cloud-based attendance management system currently used in a junior school. The platform enables authorized teachers to securely record class attendance while administrators manage access and attendance schedules. Built with Google Apps Script and Google Sheets, the system automates attendance recording and provides centralized monitoring.

  • Security and Risk Mitigation
    • Controlled access ensures only authorized staff can submit attendance
    • Tamper-evident logging prevents manipulation of records
    • Audit-ready centralized data for administrators
    Technologies: Google Apps Script, Google Sheets, Cloud Automation
    Status: Live System (Currently in Use)

ScanSISO- QR-Based Staff Authentication & Attendance System

A QR-based staff sign-in/sign-out system replacing manual attendance books. Teachers scan a unique QR code to log arrival and departure, with instant confirmation and administrative oversight. Admins can define time-bound access windows to ensure attendance integrity.

  • Security and Risk Mitigation
    • Time-restricted access prevents fraudulent check-ins/check-outs
    • Unique QR codes prevent token reuse and unauthorized access.
    • Centralized logging provides tamper-evident staff attendance records.
    Technologies: Google Apps Script, Google Sheets, QR code integration
    Status: Live System (Currently in Use)
